Coding

chess.com Data Scraper

Part of a research project, I built a script in Python to collect pulbic data of different users from chess.com. The script was automatically ran once a day. The program ran on AWS Lambda with storage intergration with AWS S3.

Instagram Data Scraper Scraper

This project was another Python-based web scraper that utilized Postman to generate requests, generate an access token for Instagram users, and retrieve metadata like date, likes, and location from a given user's posts.

RobinHood Data Scraper Scraper

This project was a Python-based application that used Robinhood's (now defunct) API to login, make routine operations, and catalog historical data of the account such as trades performed, prices of stock, and buying and selling actions.

Education Game in Blender

As part of my Computers for Learning/Human-Computer Interaction, I contributed towards an educational game, in which a player explores several different biomes, learning about the varying ecosystems. The process involved coding in C#, game design in Unity, and custom modeling in Blender.
